GraphChecker tester

This is a small helper application for GraphChecker that allows testing checks or the Graph UI plugin, avoiding the need to actually set up a Moodle server. It allows you to draw a graph and pick some checks, and see if they pass or fail.

Press Ctrl+Alt+M (or Ctrl+Alt+Shift+M) to see the JSON-encoded version of the graph and checks.

Note: If you are editing the JavaScript code for the editor, but your changes are not showing up after refreshing, make sure to disable the browser cache. (The JS files are loaded asynchronously, which means that even force-refreshing with Ctrl+F5 may not clear the cache.)
